Generative Modeling for Inclusive Facial Image Creation

Created a class-conditioned diffusion model with U-Net, generating diverse facial images from the Fairface Dataset, overcoming resolution constraints for high-quality outputs.

Project Overview

This project focuses on generating diverse and inclusive facial images using diffusion-based generative modeling techniques. The primary goal is to create synthetic facial images that blend features from three distinct ethnic groups: Orientals, Indians, and White Europeansâ€”ensuring balanced and fair representation in AI-generated content.

We built a class-conditioned diffusion model using a U-Net architecture enhanced with class embeddings. The model was trained on a well-prepared dataset sourced from the Fairface dataset, which was augmented and categorized to include equal representation from all three ethnic groups. The system allows users to generate realistic facial images for any combination of groups, highlighting the power of controlled synthetic image creation. The project demonstrates advanced techniques in dataset preparation, model training, and feature blending. Despite challenges such as resolution limitations and long training cycles, the model successfully generated high-quality, inclusive facial images that reflect diverse human identities. This work lays the foundation for further research into ethical and representative generative AI.
